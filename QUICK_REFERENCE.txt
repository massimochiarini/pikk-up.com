  STEP 1: RUN DATABASE MIGRATION (ONE TIME)
═══════════════════════════════════════════════════════════════════

  1. Open Supabase Dashboard: https://app.supabase.com
  2. Select your project
  3. Click "SQL Editor" in left sidebar
  4. Click "New query"
  5. Copy/paste: Database/QUICK_START_PUBLIC_BOOKINGS.sql
  6. Click "Run"
  7. Wait for success message

  ✅ Done! Database is ready.

═══════════════════════════════════════════════════════════════════
  STEP 3: TEST THE FEATURE
═══════════════════════════════════════════════════════════════════

  As Instructor:
  1. Login to web app
  2. Go to "My Games"
  3. Click "Copy Booking Link" on any session
  4. Paste link somewhere (email, notes, etc.)

  As Client (use incognito/private browser):
  1. Paste the link in browser
  2. Fill out the form
  3. Click "Reserve My Spot"
  4. See confirmation screen

  Verify:
  1. Go back to instructor view
  2. Open the session details
  3. See the guest in "Who's going" section

  ✅ Done! Feature is working.
